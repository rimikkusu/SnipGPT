# SnipGPT

> Capture your screen. Get instant AI answers. No friction.

SnipGPT is a lightweight desktop tool that lets you snip any region of your screen and receive AI-powered responses in a clean overlay — right where you need them.

---

## Features

**AI Providers**
- OpenAI (GPT models)
- Google Gemini

**Screenshot Modes**
- App-native selection (custom snipping)
- Windows Snipping Tool (`Shift + Win + S`)
- Both modes combined

**Customizable Overlay**
- Position: top/bottom, left/right
- Background: black, white, or none
- Adjustable opacity
- Custom selection box appearance

**Usability**
- Global hotkey support
- Clean, minimal settings UI
- Fast and lightweight

## Setup

1. Download and extract the application files
2. Set your API key(s) as environment variables:

   ```
   OPENAI_API_KEY=your_openai_key
   GEMINI_API_KEY=your_gemini_key   # optional
   ```

3. Run the application

---

## Usage

1. Launch SnipGPT and enable listening
2. Press `Ctrl + Alt + S` (default hotkey) to capture a screen region
   — or use `Shift + Win + S` if Windows Snipping mode is enabled
3. The AI response appears instantly in the overlay

---

## Notes

- Some AI models do not support image input — check your provider's documentation
- API usage may incur costs depending on your plan and provider
- Ensure your API keys are valid and correctly set before first use
